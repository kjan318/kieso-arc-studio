## How to use it
When creating a new sub-project in the `projects/` directory, you must automatically generate a `README.md` file in the root of that new project folder using the exact template below. 

### Constraints:
- Do not modify the structure of the template.
- Fill in the bracketed `[ ]` information based on the code you just generated.
- Categorize the "Domain" accurately based on the nature of the project.
